# رگرسیون خطی چندگانه

در این بخش، یک مثال واقعی از رگرسیون خطی چندگانه با استفاده از پایتون ارائه شده است.
هدف این تمرین، آشنایی عملی با مدل رگرسیون خطی چندگانه، کمینه‌سازی مجموع مربعات خطا (RSS)
و تفسیر ضرایب مدل است.

## فایل‌ها
- `MultipleLinearRegression.ipynb`  
  نوت‌بوک آموزشی شامل بارگذاری داده‌ها، برازش مدل رگرسیون خطی و رسم نمودارها.

- `Volve_CleanedData.csv`  
  دیتاست نمونه مورد استفاده در این تمرین.

## نحوه اجرا
1. نوت‌بوک را مستقیماً در گوگل کولب باز کنید.
2. سلول‌ها را به‌ترتیب اجرا نمایید.
