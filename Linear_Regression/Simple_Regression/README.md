# رگرسیون خطی ساده

در این بخش، یک مثال ساده از رگرسیون خطی با استفاده از پایتون ارائه شده است.
هدف این تمرین، آشنایی عملی با مفهوم خط رگرسیون، کمینه‌سازی مجموع مربعات خطا (RSS)
و تفسیر ضرایب مدل است.

## فایل‌ها
- `SimpleLinearRegression.ipynb`  
  نوت‌بوک آموزشی شامل بارگذاری داده‌ها، برازش مدل رگرسیون خطی و رسم نمودارها.

- `SimpleLR.csv`  
  دیتاست نمونه مورد استفاده در این تمرین.

## نحوه اجرا
1. نوت‌بوک را مستقیماً در گوگل کولب باز کنید.
2. سلول‌ها را به‌ترتیب اجرا نمایید.
3. خروجی‌ها شامل نمودار داده‌ها، خط رگرسیون و معیارهای آماری هستند.
